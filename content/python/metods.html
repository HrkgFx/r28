String Methods - Joining & Splitting

.count()

Sequence types in Python implement .count() method. This method is used to determine the number of occurences of designated object in a sequence:

s='AbbcabbA'
s.count('a')
results is 1
str.join(iterable)

We want to use join() method to concatenate collection of strings. join() method is a highly optimized equivalent to string concatenation. String (str) on which the operation is performed is actually a delimiter which is inserted between the items contained in the iterable argument.

', '.join(['1','2','3'])
results is '1, 2, 3'
str.split(sep=None, maxsplit=-1)

We could say that split() method is opposite to join() method. Join() glues together a collection of strings using a delimiter, meanwhile split() chops the string into a list of strings.

'1 2 3'.split()
'1,2,3'.split(',')
results is ['1','2','3']
results is ['1','2','3']
str.splitlines([keepends])

splitlines() method splits the string on line end delimiters. There are multiple characters used to mark the end of a line (e.g. \n,\r)

'This is the first line\nThis is the second line'.splitlines()
'This is the first line\nThis is the second line'.splitlines(keepends = True)
results is ['This is the first line','This is the second line']
results is ['This is the first line\n',This is the second line']
str.partition(separator)

The partition() method is used idiomatically to get rid of separators, which are stored in dummy variable (often for the name of dummy variable is used underscore symbol: _. Dummy variable is not later used in the code.

'Big&Small'.partition('&')
'up-to-date'.partition('-')
results is ('Big', '&'', 'Small')
results is ('up', '-', 'to-date')

String Methods - Replacing & Removing

str.replace(what,forwhat)

Replaces a substring for another substring inside the operated string.

'Race'.replace('R','F')
'Race'.replace('b','c')
results is 'Face'
results is 'Race'
str.strip(chars)

The chars argument is a string representing all characters to be removed from the end or the beginning of the string:

' too many spaces '.strip()
'www.engeto.com'.strip('.cmow')
'Entire line\n'.strip('n')
'Entire line\n'.strip('\n')
results is 'too many spaces'
results is 'enget'
results is 'Entire line\n'
results is 'Entire line'
str.lstrip(chars)

The chars argument is a string representing all characters to be removed from the beginning of the string:

"www.engeto.com".lstrip('.cmow')
'\tEntire line\n'.lstrip()
 
' too many spaces '.lstrip()
results is 'engeto.com'
Characters \t or \n are considered whitespace as well as ' ': results is 'Entire line\n'
results is 'too many spaces '
str.rstrip(chars)

The chars argument is a string representing all characters to be removed from the end of the string:

"www.engeto.com".rstrip('.cmow')
'\tEntire line\n'.rstrip()
 
' too many spaces '.rstrip()
results is 'www.enget'
Characters \t or \n are considered whitespace as well as ' ': results is '\tEntire line'
results is ' too many spaces'

String Methods - Finding Strings

str.startswith(substr[,start[,end]])

We are asking, whether a specific string starts with a specific character respectively sequence of characters

  • substr - Substring or sigle character we are looking for
  • start - Integer specifying the index at which we want the check to begin
  • end - Integer specifying the index at which we want the check to end
'Python'.startswith('P')
'Python'.startswith('Py')
'Python'.startswith('t',2)
'Python'.startswith('t',2,4)
'Python'.startswith('t',1)
'Python'.startswith('th',2)
results is True
results is True
results is True
results is True
results is False
results is True
str.endswith(substr[,start[,end]])

Similar to str.startswith with the difference, this method is checking, whether a string ends with substring substr.

str.find(sub[, start[, end]])

Looks for the first occurence of a substring in a string. We can optionally specify the range, where the lookup should begin and end. If the substring was not found, value -1 is returned. Find is a very expensive operation. Compare with str.index(), which raises error instead of returning -1 if value not found.

  • sub - Substring or single character we are looking for
  • start - Integer specifying the index at which we want the check to begin
  • end - Integer specifying the index at which we want the check to end
'anamgram'.find('a')
'anamgram'.find('a',1)
'anamgram'.find('m')
'anamgram'.find('h')
results is index 0
results is index 2
results is index 3
results is index -1
str.rfind(substr[,start[,end]])

The rfind() method works the same as the find() method with the difference that the search begins from the right side and the index returned is the first position from the end, where the substring has been found

  • substr - Substring or single character we are looking for
  • start - Integer specifying the index at which we want the check to begin
  • end - Integer specifying the index at which we want the check to end
'anagram'.rfind('a')
'anagram'.rfind('a',3)
'anagram'.rfind('a',5)
'anagram'.rfind('a',6)
results is index 5
results is index 5
results is index 5
results is index -1
